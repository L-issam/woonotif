Demande de Suppression des Données - WooNotif

ℹ️ Informations importantes

WooNotif stocke toutes vos données uniquement sur votre appareil. Aucune donnée n'est stockée sur nos serveurs ou sur des serveurs distants.

Comment supprimer vos données

Pour supprimer toutes vos données de l'application WooNotif, vous avez deux options :

1 Méthode 1 : Désinstaller l'application

La méthode la plus simple pour supprimer toutes vos données :

  1. Allez dans les paramètres de votre appareil Android
  2. Recherchez "WooNotif" dans la liste des applications
  3. Appuyez sur "Désinstaller"
  4. Confirmez la désinstallation

Résultat : Toutes les données stockées localement seront supprimées définitivement.

2 Méthode 2 : Réinitialiser les données de l'application

Si vous souhaitez garder l'application mais supprimer vos données :

  1. Allez dans les paramètres de votre appareil Android
  2. Recherchez "WooNotif" dans la liste des applications
  3. Appuyez sur "Stockage" ou "Espace de stockage"
  4. Appuyez sur "Effacer les données" ou "Réinitialiser"
  5. Confirmez l'action

Résultat : Toutes les données de l'application seront supprimées, mais l'application restera installée.

⚠️ Données supprimées

Lors de la suppression, les données suivantes seront supprimées :

  • URL de votre boutique WooCommerce
  • Consumer Key (Clé API)
  • Consumer Secret (Secret API)
  • Préférences de notification
  • Dernier ID de commande connu
  • Toutes les autres données stockées localement

✅ Données conservées

Aucune donnée n'est conservée après la suppression. Toutes les données sont stockées uniquement sur votre appareil, donc une fois supprimées, elles le sont définitivement.

⚠️ Durée de conservation

Durée de conservation : Les données sont conservées jusqu'à ce que vous les supprimiez (via désinstallation ou réinitialisation). Il n'y a pas de suppression automatique.

Après suppression : Les données sont supprimées immédiatement et définitivement. Aucune donnée n'est conservée après la suppression.

Données partagées avec des tiers

Google AdMob : Si vous avez utilisé l'application, Google AdMob peut avoir collecté des données anonymes pour les publicités. Ces données sont gérées selon la Politique de confidentialité de Google.

Important : Nous n'avons aucun contrôle sur les données collectées par Google AdMob. Pour supprimer ces données, vous devez contacter Google directement ou utiliser les paramètres de votre compte Google.

Confirmation de suppression

Après avoir désinstallé l'application ou réinitialisé ses données, toutes vos données locales sont supprimées immédiatement. Il n'y a pas de période de conservation supplémentaire, car nous ne stockons aucune donnée sur nos serveurs.
