Politique de Confidentialité - WooNotif

Dernière mise à jour : 30 novembre 2024

Introduction

WooNotif ("nous", "notre", "l'application") respecte votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos informations lorsque vous utilisez notre application mobile.

Légitimité de l'application

WooNotif a été développée pour répondre à un besoin réel : permettre aux collaborateurs de suivre les commandes WooCommerce sans installer de plugins lourds comme Jetpack. Cette application est une solution testée en conditions réelles, créée pour simplifier la gestion des commandes depuis un appareil mobile.

Informations que nous collectons

⚠️ Important : Nous ne stockons aucune information sur un serveur distant, que ce soit chez nous ou chez n'importe quel tiers. Toutes les données collectées sont stockées uniquement sur votre appareil dans le stockage local sécurisé.

Informations d'authentification

  • URL de votre boutique WooCommerce : Nécessaire pour se connecter à votre API
  • Consumer Key (Clé API) : Identifiant d'authentification WooCommerce
  • Consumer Secret (Secret API) : Clé secrète d'authentification WooCommerce

Ces informations sont stockées uniquement sur votre appareil dans le stockage local sécurisé de l'application (AsyncStorage).

Données des commandes

  • Informations des commandes WooCommerce (numéro, date, client, montant, statut)
  • Ces données sont récupérées depuis votre boutique WooCommerce via l'API REST

Données techniques

  • Permissions de l'appareil (caméra pour scanner les QR codes)
  • Préférences de notification
  • Données de diagnostic anonymes (en cas d'erreurs)

Comment nous utilisons vos informations

Utilisation des données

  • ✓ Affichage des commandes : Pour vous permettre de visualiser vos commandes WooCommerce
  • ✓ Notifications : Pour vous alerter des nouvelles commandes (si activé)
  • ✓ Authentification : Pour se connecter à votre API WooCommerce de manière sécurisée

Ce que nous ne faisons PAS

  • ✗ Nous ne collectons PAS vos données en dehors de votre appareil
  • ✗ Nous ne vendons PAS vos données à des tiers
  • ✗ Nous n'envoyons PAS vos identifiants à nos serveurs
  • ✗ Nous n'accédons PAS à votre boutique sans votre autorisation explicite

Stockage et sécurité

Stockage local

Toutes vos informations sont stockées uniquement sur votre appareil :

  • Les identifiants API sont chiffrés et stockés localement
  • Aucune donnée n'est synchronisée avec nos serveurs
  • Vous pouvez supprimer toutes les données en désinstallant l'application

Sécurité

  • Connexions HTTPS uniquement (obligatoire)
  • Authentification Basic Auth sécurisée
  • Stockage local chiffré par le système d'exploitation
  • Aucune transmission de données sensibles en clair

Permissions de l'application

Caméra

  • Usage : Scanner les QR codes contenant vos clés API WooCommerce
  • Stockage : Aucune image n'est sauvegardée
  • Accès : Uniquement lorsque vous activez le scanner QR

Notifications

  • Usage : Vous alerter des nouvelles commandes
  • Contrôle : Vous pouvez activer/désactiver à tout moment dans les paramètres
  • Données : Aucune donnée personnelle n'est incluse dans les notifications

Internet

  • Usage : Se connecter à votre boutique WooCommerce via l'API REST
  • Sécurité : Connexions HTTPS uniquement

Vos droits

Vous avez le droit de :

  • ✓ Accéder à vos données stockées localement
  • ✓ Supprimer toutes vos données (désinstaller l'application)
  • ✓ Révoquer l'accès à votre boutique à tout moment
  • ✓ Désactiver les notifications
  • ✓ Demander des informations sur vos données

Pour exercer ces droits, vous pouvez :

  • Supprimer l'application de votre appareil
  • Réinitialiser les données de l'application dans les paramètres système
  • Nous contacter (voir coordonnées ci-dessous)

Cookies et technologies de suivi

L'application n'utilise pas de cookies ni de technologies de suivi tierces. Toutes les communications se font directement entre votre appareil et votre boutique WooCommerce.

Publicités (Google Ads)

Si vous avez activé les publicités dans l'application :

  • Google AdMob peut collecter des données anonymes pour personnaliser les publicités
  • Ces données sont gérées selon la Politique de confidentialité de Google
  • Vous pouvez désactiver la personnalisation des publicités dans les paramètres de votre appareil

Partage avec des tiers

Nous ne partageons vos données avec aucun tiers, sauf :

  • Google AdMob : Uniquement pour les publicités (données anonymes)
  • Votre boutique WooCommerce : Directement via l'API REST (données que vous contrôlez)

Conservation des données

Vos données sont conservées :

  • Sur votre appareil : Jusqu'à ce que vous supprimiez l'application ou réinitialisiez les données
  • Aucune donnée n'est conservée sur nos serveurs (nous n'en avons pas)

Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront signalées par :

  • Une notification dans l'application
  • Une mise à jour de la date "Dernière mise à jour"
  • Une nouvelle version de l'application

Conformité légale

Cette application est conforme au :

  • RGPD (Règlement Général sur la Protection des Données) - UE
  • CCPA (California Consumer Privacy Act) - Californie
  • Loi sur la protection des données - Canada

Consentement

En utilisant WooNotif, vous acceptez cette politique de confidentialité. Si vous n'acceptez pas cette politique, veuillez ne pas utiliser l'application.

Note importante : Cette application fonctionne comme un client pour votre boutique WooCommerce. Toutes les données restent sous votre contrôle et sont stockées uniquement sur votre appareil. Nous n'avons aucun accès à vos données.
