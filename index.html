WooNotif

Suivez vos commandes WooCommerce en temps réel, directement depuis votre smartphone.

Qu'est-ce que WooNotif ?

WooNotif est une application mobile pensée pour les propriétaires de boutiques WooCommerce qui veulent suivre leurs commandes sans ouvrir un ordinateur.

  • Affichage des commandes récentes WooCommerce
  • Notifications (si activées) pour les nouvelles commandes
  • Vue rapide du client, du montant et du statut
  • Connexion directe à votre propre boutique via l'API REST WooCommerce

L'application ne stocke aucune donnée sur des serveurs distants : tout reste sur votre appareil.

Comment obtenir vos clés WooCommerce (CK/CS)

1. Se connecter à votre interface WordPress

  • Connectez-vous à l'administration de votre site WordPress (compte administrateur).
  • Assurez-vous que le plugin WooCommerce est installé et actif.

2. Accéder aux clés API WooCommerce

  • Dans le menu WordPress, allez dans WooCommerce > Paramètres.
  • Ouvrez l’onglet Avancé.
  • Cliquez sur REST API puis sur Ajouter une clé (ou "Add key").

3. Créer une clé en mode lecture seule

  • Donnez un nom à la clé (par exemple : WooNotif).
  • Sélectionnez l’utilisateur propriétaire de la boutique.
  • Dans Autorisations, choisissez Lecture uniquement.
  • Cliquez sur Générer une clé API.

4. Récupérer les identifiants

  • Copiez la Consumer Key (ck_...).
  • Copiez la Consumer Secret (cs_...).
  • Renseignez ces deux valeurs, ainsi que l’URL de votre boutique, dans l’application WooNotif.

Pour plus de sécurité, nous recommandons d’utiliser un compte dédié avec uniquement les permissions nécessaires.

Fonctionnement technique et confidentialité

  • Connexion à votre boutique uniquement via HTTPS.
  • Identifiants API stockés uniquement sur votre appareil (AsyncStorage).
  • Aucune donnée envoyée à nos propres serveurs (nous n’en avons pas).
  • Publicités gérées par Google AdMob (voir leur politique de confidentialité).

Tous les détails sont expliqués dans notre politique de confidentialité.

Évolutions prévues

Nous prévoyons d’ajouter prochainement :

  • Gestion de plusieurs boutiques WooCommerce dans la même application à venir
  • Filtres avancés sur les commandes (statuts, montants, dates)
  • Options supplémentaires de personnalisation des notifications

Ces évolutions seront annoncées dans les futures mises à jour de l’application.
